# 说明

当前项目是一个基于工作流构建的一个自动化个人助手平台。重点:中文项目。

## 路径说明

com/zaw:
    -common: 公共包
    -workflow: 工作流业务模块
        -controller 接口层
        -entity 数据表层
        -enums 枚举
        -mapper 映射工具类
        -repository jpa repository类
        -service 服务类层(除非抽象设计，不需要定义interface)
        -web web DTO

## 具体要求

### controller

接口层采取restful风格

## 代码要求

接口层注明接口中文注释
服务类层在if else必须注明中文注释
方法上必须加入中文注释
方法内有分步处理时，需给每一步骤添加注释
